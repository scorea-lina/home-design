# Troubleshooting

## Common issues

### "Missing Access" in Discord
- Cause: bot account not allowlisted for the channel or missing Discord permissions.
- Fix: update OpenClaw discord allowlist + ensure View/Send/History perms.

### QA alias looks stale
- Try hard refresh (Cmd/Ctrl+Shift+R) or incognito.

### Cron UI crashes
- Ensure API returns safe defaults for missing fields.
